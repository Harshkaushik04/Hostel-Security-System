commands to run sfu v1:

on mediaMTX server:
go in mediaMTX folder and run ./mediamtx

open camera source:
ffmpeg -f v4l2 -framerate 30 -video_size 1920x1080 -i /dev/video0 -pix_fmt yuv420p -c:v libx264 -preset ultrafast -tune zerolatency -profile:v baseline -level 4.0 -g 30 -b:v 4M -flags +global_header -f rtsp -rtsp_transport tcp rtsp://localhost:8554/camera1

run sfu server:
go into sfu_server folder and run using "npm run dev"

run incerceptor between mediaMTX server and sfu server:

ffmpeg -rtsp_transport tcp -i rtsp://localhost:8554/camera1 -c:v copy -payload_type 112 -ssrc 2222 -pkt_size 1200 -f rtp rtp://127.0.0.1:42317

run frontend on port 5173:
go into dummy_frontend folder and run "npm run dev"
